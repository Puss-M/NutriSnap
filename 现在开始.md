# 🎯 现在需要做什么？

## 第一步：获取 Supabase 密钥

1. 打开浏览器，访问: https://supabase.com/dashboard
2. 登录或创建账号
3. 点击「New Project」创建新项目（或选择已有项目）
4. 等待项目初始化完成（~2 分钟）
5. 进入项目后，点击左侧菜单 **Settings** → **API**
6. 复制两个值：
   - **Project URL** (类似: https://xxxxx.supabase.co)
   - **anon public** key (一长串以 eyJ 开头的字符串)

## 第二步：获取 Silicon Flow API 密钥

你之前的对话中用过 Silicon Flow，应该已经有 API key 了。

如果没有:

1. 访问: https://siliconflow.cn/
2. 登录/注册
3. 获取 API 密钥 (以 sk-开头)

## 第三步：更新 .env.local 文件

1. 打开项目文件夹中的 `.env.local` 文件
2. 用你的真实密钥替换 placeholder:

```bash
NEXT_PUBLIC_SUPABASE_URL=https://你的实际url.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJ你的实际key...
SILICON_FLOW_API_KEY=sk-你的实际key
```

3. 保存文件

## 第四步：设置数据库

1. 在 Supabase 控制台，点击左侧 **SQL Editor**
2. 点击「New Query」
3. 打开项目中的 `supabase-schema.sql` 文件，复制全部内容
4. 粘贴到 SQL Editor 并点击「Run」
5. 等待执行完成（应该看到 "Success"）

## 第五步：创建存储桶

1. 在 Supabase 控制台，点击左侧 **Storage**
2. 点击「New bucket」
3. 名称填写: `food-images`
4. **重要**: 关闭「Private bucket」选项（选择 Public）
5. 点击「Create bucket」

## 第六步：重启服务器

在终端中:

1. 按 `Ctrl+C` 停止当前服务器
2. 运行 `npm run dev` 重新启动
3. 刷新浏览器页面

---

**完成后，你会看到主页面而不是这个设置提示！**

如果遇到问题，检查 QUICKSTART.md 获取详细说明。
